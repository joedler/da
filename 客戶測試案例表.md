# 客戶測試案例表

| 編號 | 測試對象類型 | 測試項目 | 測試方式 | 預期結果 | 結果 | 備註 |
|---|---|---|---|---|---|---|
| 1 | 學生 | 首次驗證 | 輸入姓名 + 學號 | 綁定成功並顯示個人資料 |  |  |
| 2 | 老師 | 首次驗證 | 輸入姓名 + 手機末三碼 | 綁定成功並顯示個人資料 |  |  |
| 3 | 素食者 | 素食註記 | 查看個人資料 | 顯示正確素食註記 |  |  |
| 4 | 同房多人 | 同房人員 | 查看房間編組與同房人員 | 顯示正確同房名單 |  |  |
| 5 | 房號未公告 | 房號顯示 | 查看我的房號 | 顯示 `尚未公告` |  |  |
| 6 | 已綁定使用者 | 再次開啟 | 關閉 LIFF 後重新開啟 | 不需重新輸入，直接顯示個人資料 |  |  |
| 7 | 已綁定使用者 | 解除綁定 | 點選解除綁定 | 回到首次驗證畫面 |  |  |
| 8 | 錯誤資料 | 學號錯誤 | 姓名正確、學號錯誤 | 顯示驗證失敗 |  |  |
| 9 | 錯誤資料 | 姓名不存在 | 輸入不存在姓名 | 顯示查無此姓名 |  |  |

## 回報格式

若測試發現問題，請使用下列格式回報：

```text
測試者：
身份：學生 / 老師
輸入方式：姓名+學號 / 姓名+手機末三碼
問題欄位：
畫面顯示：
正確內容：
截圖：
```
