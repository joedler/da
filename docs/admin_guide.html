領隊如何綁定領隊權限？

1

在 LINE 輸入關鍵字

對小幫手的 LINE 對話視窗輸入 「總覽」 或 「管理」，小幫手會回傳一張領隊專用的卡片。

2

輸入姓名與手機末三碼

點擊卡片上的「開啟總覽」，在跳出的驗證頁面填入「姓名」與「手機末三碼」，系統會比對領隊權限表完成驗證。

3

綁定完成，之後免密碼登入

驗證成功後，這支 LINE 帳號就綁定了領隊身份。之後不管何時開啟總覽，都不需要再輸入任何資料。

後台有哪些功能可以用？

即時分車清查點名

「分車分頁」 會自動以領隊所在的「服務車次」（如 A 車）置頂顯示。名單中會直接顯示學生和老師是否已經完成「 LINE 綁定」。未綁定者系統會標示為「未綁定」，方便隨車領隊在車上口頭宣導及催促綁定。

全局即時資料模糊搜尋

管理頁面上方設有「智能搜尋欄」。輸入姓名、班級、車次、房號、桌號甚至是「素食」等關鍵字，名單隨即毫秒級縮小篩選範圍，方便領隊現場快速答覆團員疑問。

桌次與素食對照

「桌次分頁」 以桌號排序，列出每一桌的團員名字。若團員註記為素食，名字後方會自動帶上紅色的「（素）」標誌，方便餐廳上菜與領隊核對人數。

現場常见狀況處理方式

狀況一

學生因個人因素臨時更換房間

領隊處理方式：

  1. 手機或電腦開啟雲端 Google Sheets 的 people 工作表
  2. 搜尋該名學生姓名
  3. 修改 roomGroup（房間編組）與 firstDayRoomNo / secondDayRoomNo（房號）
  4. 學生在 LINE 點選「我的資訊」，即會即時展示新房間與室友！
狀況二

團員反映「老師，我被綁錯了」

領隊處理方式：

  1. 方法 A（師生自行）： 指導師生進入系統「我的資訊」分頁 → 拉到最底 → 點選「解除綁定」
  2. 方法 B（領隊強制）： 在 Sheets people 搜尋姓名，將「 LINE使用者ID」與「綁定時間」欄位清除（留空）
  3. 團員即可重新驗證綁定
狀況三

飯店臨時通知某幾間房號更動

領隊處理方式：

  1. 開啟 Sheets people 工作表
  2. 找到對應房間編組（例如男1）
  3. 將受影響學生的房號格直接改掉
  4. 不影響其他未更動的人，資訊為實時動態載入
狀況四

隨隊老師想看整輛車的素食名單

領隊處理方式：

  1. 領隊在自己的 LINE 點選「開啟總覽」進入後台
  2. 點選右上角搜尋放大鏡
  3. 輸入「素」或「素食」關鍵字
  4. 將手機拿給隨車老師，一秒看到筛選後名單！
