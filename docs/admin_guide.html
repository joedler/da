領隊權限綁定三步驟

1

對話視窗輸入關鍵字

對 LINE 官方帳號對話視窗輸入 「總覽」 或 「管理」，Bot 隨即彈出專屬領隊後台 Flex 點擊卡片。

2

姓名與手機末三碼驗證

點擊「開啟總覽」，在跳出的授權網頁輸入「姓名」與「手機末三碼」，系統將比對 admins 權限表完成驗證。

3

鎖定 LINE 帳號自動登入

驗證完成後，此 LINE 帳號已永久綁定領隊身分。之後任何時候點開 LINE 均可免密碼一秒開啟後台管理。

領隊總覽後台核心功能

即時分車清查點名

「分車分頁」 會自動以領隊所在的「服務車次」（如 A 車）置頂顯示。名單中會直接顯示學生和老師是否已經完成 **「LINE 綁定」**。未綁定者系統會標示為「未綁定」，方便隨車領隊在車上口頭宣導及催促綁定。

全局即時資料模糊搜尋

管理頁面上方設有 **「智能搜尋欄」**。輸入姓名、班級、車次、房號、桌號甚至是「素食」等關鍵字，名單隨即毫秒級縮小篩選範圍，方便領隊現場快速答覆團員疑問。

桌次與素食對照

「桌次分頁」 以桌號排序，列出每一桌的團員名字。若團員註記為素食，名字後方會自動帶上紅色的 **「（素）」** 標誌，方便餐廳上菜與領隊核對人數。

現場常見突發狀況應變

狀況一

學生因個人因素臨時更換房間

領隊處理方式：
直接用手機瀏覽器或電腦開啟雲端 Google Sheets 的 `people` 工作表，搜尋該名學生姓名，並修改其 `roomGroup` (房間編組) 與 `firstDayRoomNo` / `secondDayRoomNo` (房號)。修改完成後，請學生再次打開 LINE 點選「我的資訊」，即會一秒顯示新房間與新室友！

狀況二

團員反映「老師，我被綁錯了」

領隊處理方式：
1. 可指導該師生直接點進系統的「我的資訊」分頁，拉到最底點選「解除綁定」。
2. 或者，領隊也可以在雲端 Sheets 的 `people` 中搜尋其姓名，把該列的 `LINE使用者ID` 與 `綁定時間` 刪除（留白），即可完成強制解除綁定，讓團員重做驗證。

狀況三

飯店臨時通知某幾間房號更動

領隊處理方式：
在 Sheets 的 `people` 找到相對應的房間編組（例如男1），將受影響學生的房號格直接改掉。不影響其他未更動的人，資訊為實時動態載入。

狀況四

隨隊老師想看整輛車的素食名單

領隊處理方式：
領隊直接在自己的 LINE 點選「開啟總覽」，進入後點選右上角的搜尋放大鏡，輸入例如 **「素」** 或 **「桌次」**，就可以直接將手機拿給隨車老師看一秒篩選後的名單，快速又專業！
