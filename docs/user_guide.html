3秒快速綁定三步驟

1. 加入好友並開啟連結

點擊 LINE 邀請連結，或者掃描小幫手二維碼，系統會自動在 LINE 軟體中開啟驗證網頁。

2. 填寫姓名與驗證資訊

學生請填入「中文姓名」與「完整學號」；老師請填入「姓名」與「手機末三碼」，確認無誤後點擊「驗證並綁定」。

3. 綁定成功即刻查詢

綁定完成後，之後只要在 LINE 點選連結或輸入任何文字（如「查詢」），就會一秒彈出你的個人小卡片，隨時隨地即時查詢！

車次與桌號

隨隊聯絡窗口

整合隨車領隊、旅行社窗口、學校護理師電話，一鍵直撥撥號。

精準三日行程表

動態時間軸行程，載明集合時間、住宿地址與各項活動提醒。
